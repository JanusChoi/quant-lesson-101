# 概念卡：因子暴露

## 一句话定义
一只股票在某个风险因子上的“敏感度”或“载荷”。

## 详解
在结构化风险模型里，股票的风险不是凭空出现的，而是来自它对各类共同因子的暴露。行业因子的暴露通常是 0 或 1；而规模、波动率、动能等风格因子的暴露，通常会做标准化处理，使全市场均值为 0、标准差为 1。

## 原文出处
- **书籍**：《主动投资组合管理》
- **章节**：第3章 风险

## 交易员视角
当你说“这个组合偏小盘、偏高波动、偏价值”，本质上就是在说它的因子暴露偏离了基准。
