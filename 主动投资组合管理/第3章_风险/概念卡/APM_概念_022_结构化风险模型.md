# 概念卡：结构化风险模型

## 一句话定义
用少量共同因子加上每只股票自己的特异风险，来解释和预测整个股票池的协方差结构。

## 详解
书中 `3-16` 的核心思想，是把单只股票的超额收益拆成“共同因子部分 + 特异部分”：

`r_n = Σ_k (X_{n,k} · b_k) + u_n`

其中：
- `r_n`：股票 `n` 在考察期内的超额收益率
- `X_{n,k}`：股票 `n` 对因子 `k` 的暴露
- `b_k`：因子 `k` 在该期实现的收益率
- `u_n`：股票 `n` 的特异收益率，也就是不能被共同因子解释的那部分

这就是结构化风险模型的收益率结构方程。它的价值在于降维：原本几千只股票的协方差问题，可以转化为少量因子协方差加上特异风险的问题，既更稳健，也更容易更新。

书里还特别强调了两个关键点：
- 暴露 `X_{n,k}` 必须在期初已知，所以风险模型使用的是先验因子
- 在时间顺序上，暴露是期初观测到的，因子收益率和特异收益率是在期末实现的

在进一步假设“特异收益率与因子收益率不相关，且不同股票之间的特异收益率互不相关”后，就能从 `3-16` 推到后面的协方差结构式 `3-17`。

## 原文出处

## 交易员视角
不要把风险模型理解成一张巨大的“股票两两相关表”。更实用的理解是：你的组合收益，先被拆成行业、规模、波动率、价值这些共同维度，再加上一点点个股自己的特异部分。
